Quick Start

The app is 100% static — no build step, no npm. Camera requires HTTPS or localhost.

Option A — Python (recommended)

  1. Navigate to the project folder in your terminal
  2. Run: python3 -m http.server 8080
  3. Open http://localhost:8080 in your browser

Option B — Node.js

npx serve .

Option C — Deploy as PWA (install on phone)

  1. Drag the project folder onto netlify.com/drop
  2. Get a public HTTPS URL
  3. On iPhone Safari: Share → Add to Home Screen

Why HTTPS? Browser security rules block camera on plain file:// URLs. Localhost and HTTPS both work.

How the New UI Works

Single Filtered View

The <video> element is hidden with display:none. Only #main-canvas is displayed, already processed by the full filter pipeline. The user sees only the final filtered image at all times — no raw camera alongside it.

Camera Name Watermark

The .cam-name-overlay div sits at the bottom-left of the viewfinder. It contains three lines: a small "NOW SHOOTING" mono label, the large Bebas Neue camera name (e.g. Golden Hour), and the category. When you switch presets, the name fades out and fades back in via a 120ms opacity animation in _updateNameOverlay() in app.js.

Magnetic Tabs

The .cat-indicator div is an absolutely-positioned pill that slides over the active category tab. It is controlled by js/tabs.js which reads the tab's bounding rect relative to its wrapper and sets left and width via CSS. The transition uses a spring-like cubic-bezier (cubic-bezier(0.34, 1.56, 0.64, 1)) for the bounce effect. On hover the indicator tracks the mouse; on mouse-leave it snaps back to the active tab.

Camera Cards

Each preset card has three elements: a 3-colour swatch strip at the top (from preset.swatch), the emoji icon, and the full camera name in DM Sans. The active card gets a golden border and a small amber checkmark badge. Cards scroll horizontally with overflow-x: auto and scrollbar-width: none.

Bottom Panel Height

The panel height is controlled by the CSS variable --panel-h (default 200px, 220px on tall screens). The viewfinder's bottom value equals the same variable, so increasing it makes the viewfinder shorter and the panel taller. No JS needed.

Adding a New Preset

Open js/presets.js and add a new object to CAMERA_PRESETS:

{
  id:          'my-cam',             // unique slug, no spaces
  name:        'My Camera',          // shown as the large watermark name
  shortName:   'My Camera',          // shown on the card (same or shorter)
  category:    'Warm & Aesthetic',   // must match a cat-tab data-category
  icon:        '📷',
  swatch:      ['#f0c080','#c08040','#804020'],  // 3 hex colours for top strip
  description: 'Short description for the handover',
  filters: {
    temperature:          0.0,   // -1.0 cool → +1.0 warm
    contrast:             1.0,   // 0.5 flat  → 2.0 punchy
    saturation:           1.0,   // 0.0 B&W  → 1.6 vivid
    grain:                0.0,   // 0.0 none  → 1.0 heavy
    fade:                 0.0,   // 0.0 deep  → 0.6 matte
    blur:                 0.0,   // 0.0 sharp → 2.0 dreamy
    vignette:             0.0,   // 0.0 none  → 1.0 heavy
    chromaticAberration:  0.0,   // 0.0 none  → 3.5 strong
    gamma:                1.0    // 0.7 dark  → 1.3 bright
  }
},

No build step needed. Save the file, reload the browser — your preset appears immediately in the correct category tab.

Adding a new category tab

In index.html, add a button inside #cat-tabs:

<button class="cat-tab" data-category="My Category" role="tab" aria-selected="false">My Cat</button>

Use the exact same string in your preset's category field. The magnetic indicator will automatically include the new tab.

Customising the UI

Colours

All tokens are CSS variables at the top of styles.css:

VariableDefaultControls
--bg#0c0a08Main background
--amber#c8943aPrimary accent (tab indicator, active card border, shutter ring)
--amber-lt#e0b060Lighter accent (active card name, category name watermark)
--cream#f0e6ccPrimary text and shutter button fill
--panelrgba(14,11,9,0.94)Bottom panel glass background

Camera Name Font Size

The watermark name uses clamp(28px, 7vw, 42px) in .cno-name. Adjust the three values to make it smaller or larger.

Card Width

Cards are fixed at width: 76px in .preset-btn. Change this to make them wider or narrower.

Panel Height

Change --panel-h in :root inside styles.css. The viewfinder height adjusts automatically because it uses bottom: var(--panel-h).
